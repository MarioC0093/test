Por convenio, se establece que los valores de la prueba en enfermos son superiores que en individuos sanos. (para los ejercicos, el by default de R, etc.)

Se termina por dicotomizar los resultados de la pd en base a un treshold c.

Se(c) = TPF=P(Y ≥ c/ D=1) 1-Sp(c) = FPF=P(Y ≥ c/D=0)

curva bidimensional doned para cualquier punto posible de la prueba calculamos pa prob de positivos en sanos y negativos en enfermos.

dado un valor c, probabildaid de tener en los dos grupos un valor por encima de c.

prob de tener en los enfermos un valor por encima de c y prob de tener en los sanos un valor por encima de c.

si c \(\rightarrow\) -inf => se=1 y sp=1 si c \(\rightarrow\) +inf => se=0 y sp=0

ejemplo 2.5.

al ser distribuciones normales le pido a R la prob de tener un valor menor a c.

ej: para c=6 se(6) = prob(Y ge 6 | D=1) = 1 1-sp(6) = p(Y ge 6 | D=1) = 1

ej: para c=8.5 se(8) = prob(Y ge 6 | D=1) = 1 1-sp(8) = p(Y ge 6 | D=1) = 0.5

nos gustaria una ROC que distinguiese perfectmenete sanos y enfermos (prueba perfecta)

umbral: punto a partir del cual asigno un positivo.

si es un umbral a partir del cual hago una invasiva, que sea alta si es un umbral

df <- data.frame(c=double(),
                 `1-Sp`=double(), 
                 Se=double()) 
for (c in seq(6.5, 10.5, 0.025)){
  sp_sincirrosis <- pnorm(q = c, mean=7.5, sd=0.2, lower.tail=TRUE)
  se_concirrosis <- pnorm(q = c, mean=8.5, sd=0.6, lower.tail=FALSE)
  df <- rbind(df,c(c,1-sp_sincirrosis,se_concirrosis))
}
colnames(df) <- c('c', '1-Sp', 'Se')

ggplot(df, aes(x = `1-Sp`, y = Se)) +
  # geom_line(color = "blue", size = 1) +
  geom_point(color = "red", size = 1) +
  labs(
    title = "Curva ROC",
    x = "1 - Especificidad (Tasa de Falsos Positivos)",
    y = "Sensibilidad (Tasa de Verdaderos Positivos)"
  ) +
  theme_minimal()
